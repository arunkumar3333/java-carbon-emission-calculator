About Carbon Footprint Analyzer

The Carbon Footprint Analyzer is a web application developed to help individuals and organizations measure and analyze carbon emissions generated from transportation activities.

Transportation contributes significantly to global carbon emissions. This system allows users to upload travel data in CSV or Excel format and automatically calculates carbon emissions using predefined emission factors.

The system stores emission records in a database and provides a dashboard that visualizes emission statistics using tables and charts.

Technologies Used

  • Java (Servlets)
  • Hibernate ORM
  • MySQL Database
  • AngularJS
  • Chart.js
  • HTML / CSS
